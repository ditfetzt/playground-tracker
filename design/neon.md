# Neon — The Playground Design System

Original festival-night theme. Deep midnight aurora canvas, glass-morphism cards, electric purple-pink accents, floating particles.

## Palette

| Token | Hex | Role |
|-------|-----|------|
| `--color-background` | `#0a0612` | Deep midnight canvas |
| `--color-foreground` | `#f0e6ff` | Lavender text |
| `--color-primary` | `#b44dff` | Electric purple — buttons, progress |
| `--color-primary-foreground` | `#ffffff` | Text on primary |
| `--color-secondary` | `#1e1432a6` | Surfaces, hover |
| `--color-secondary-foreground` | `#c4b0e0` | Text on secondary |
| `--color-muted-foreground` | `#8a7a9a` | Labels, headings, meta |
| `--color-destructive` | `#ff2d95` | Hot pink delete |
| `--color-border` | `#b44dff40` | Card / input borders |
| `--color-ring` | `#ff2d95` | Focus ring |
| `--color-card` | `#2d1e4180` | Glass-card (50% opaque, blurred) |
| `--color-amber` | `#ff6b35` | Warnings, buying |
| `--color-emerald` | `#39ff8c` | Success, paid |

| Token | Value | Effect |
|-------|-------|--------|
| `--radius` | `0.875rem` (14px) | Standard rounded cards |

## Elevation

Glass-morphism: `backdrop-filter: blur(12px)` with semi-transparent backgrounds. Cards feel like frosted glass floating over the aurora. Gentle hover transitions brighten borders.

## Character

**Festival neon, glassy, ethereal.** Aurora background, floating particles, purple-pink gradients, glass-morphism cards. The most decorative of the three themes. Built for Burner festival vibes.
