# Moist — The Playground Design System

Underwater / ocean theme. Deep navy canvas with drifting cyan light rays, rising bubble particles, fluid animations. Frosted glass feels like looking through water.

## Palette

| Token | Hex | Role |
|-------|-----|------|
| `--color-primary` | `#00d4ff` | Cyan — buttons, accents, glow |
| `--color-emerald` | `#40e0d0` | Aqua turquoise — complementary accent |
| `--color-foreground` | `#e0f7fa` | White foam — primary text |
| `--color-card` | `#0d284080` | Deep blue glass |
| `--color-deep` | `#051020` | Ocean floor canvas |
| `--color-muted-foreground` | `#6090a0` | Muted aqua labels |

## Background

A deep ocean gradient flows vertically (18s `waterFlow`). Floating bubbles (12 cyan particles) drift upward. Light rays (`waterRipple` keyframes) simulate caustic underwater light shifting slowly. The whole background breathes like a submerged chamber.

## Shape

| Token | Value | Effect |
|-------|-------|--------|
| `--radius` | `1rem` (16px) | Rounded, organic, fluid |
| Cards | `blur(16px)` | Extra frosted, like thick water |

## Type

Rainbow text becomes a slow cyan-aqua gradient shimmer (6s cycle, weight 700). Section headings stay consistent. The type feels immersed, weighted, calm.

## Character

**Submerged, calm, flowing.** The cyan leak from the camp's MOIST theme. Bubbles rise. Light shifts. Cards feel like thick aquarium glass. Buttons glow cyan. Every element feels like it's underwater.
